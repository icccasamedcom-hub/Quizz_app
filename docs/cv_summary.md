# Résumé de Projet pour CV

**Plateforme de Quiz Interactive (Flask + MongoDB + Tailwind CSS)**
Conception d’une application de quiz modulaire permettant l'évaluation des connaissances sur des thématiques spécifiques (Vision, Valeurs, Mission). Ingestion dynamique de questions via scripts d'import, gestion de plusieurs niveaux de difficulté (Facile, Intermédiaire, Difficile) et calcul automatique des scores.

Déploiement d’une architecture backend robuste avec Flask, intégrant une base de données NoSQL (MongoDB) pour le stockage des questions, des utilisateurs et des tentatives de quiz. Mise en place d'une authentification sécurisée via Google OAuth (Authlib) avec gestion des sessions utilisateur.

Développement d’une interface utilisateur moderne et responsive utilisant Tailwind CSS et Jinja2, incluant un tableau de bord (Dashboard) pour le suivi des statistiques personnelles, un historique des tentatives et un classement (Leaderboard) en temps réel.

**Compétences acquises :** Flask, MongoDB (PyMongo), Authentification OAuth 2.0, Tailwind CSS, Architecture logicielle (Pattern Factory).
